> Ideas in Notes, thumbnails in Canva, titles in your head, schedule nowhere. Brodeo puts the whole pipeline on one screen, with a streak counter so you actually ship.

Consistency grows a channel, and it is the first thing to break when you are a one-person content team. Brodeo is where you capture ideas, plan the calendar, generate titles, descriptions and thumbnails with AI, and keep a publishing streak alive.

## Why it exists

The gap between "I have an idea" and "it is published" is exactly where consistency dies. Brodeo closes it by putting idea, plan, assets, and schedule in one place.

## What it does

| Feature | What it does |
|---------|--------------|
| **Idea backlog** | Dump every video idea in one place so nothing gets lost |
| **Calendar + schedule** | See what ships when; a "Next Due" clock and streak keep your cadence |
| **AI packaging** | One click drafts titles, descriptions, thumbnail text, or "generate everything" |
| **Thumbnail Studio** | Drag-drop text, 1000+ Google Fonts, template modes, AI image generation |
| **Text behind subject** | Automatic background removal for the 3-layer thumbnail look |
| **Streak tracking** | Because the entire point is to keep showing up |

## How it works

```
Capture idea  →  AI drafts title + thumbnail  →  Design in studio  →  Drop on calendar  →  Ship, streak grows
```

## Under the hood

Flask · OpenAI (text + image generation) · Google Fonts API · a background-removal step for "text behind subject" · Tailwind UI · deployed on Vercel.
